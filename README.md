# Overview

POCO는 코딩 테스트 문제를 해결해 획득한 코인으로 나만의 캐릭터를 키우고 꾸미는 코딩 육성 서비스입니다.

기존 코딩 테스트 서비스는 문제 풀이 기록과 티어 관리에 집중되어 있어 사용자가 꾸준히 코딩을 지속할 수 있는 동기부여 요소가 부족하다는 문제를 해결하고자 합니다.

POCO는 코딩을 할수록 캐릭터가 성장하고 다양한 아이템을 구매할 수 있는 보상 시스템을 제공하여, 사용자가 게임을 즐기듯 자연스럽게 코딩 습관을 형성하도록 돕는 것을 목표로 합니다.

---

# Target Audience

주요 사용자는 다음과 같습니다.

- 코딩 테스트를 공부하는 중·고등학생
- 대학생
- 취업 준비생
- 현직 개발자
- 꾸준한 코딩 습관이 필요한 사람
- 캐릭터 육성 게임을 좋아하는 사용자

---

## Core Features

### 1. 코딩 테스트 연동 및 코인 획득

- 백준, 프로그래머스 등 코딩 테스트 플랫폼과 연동
- 문제 해결 시 기본 코인 지급
- 난이도 및 풀이 시간에 따라 추가 코인 지급
- 획득한 코인은 서비스 내 재화로 사용

### 2. 캐릭터 육성 및 꾸미기

획득한 코인으로 다양한 아이템을 구매할 수 있습니다.

-  음식
-  의상
-  액세서리
-  가구

캐릭터는 음식과 운동에 따라

- 배고픔
- 행복도
- 체형

등의 상태가 변화하며 사용자의 선택에 따라 다양한 모습으로 성장합니다.

### 3. GitHub 프로필 연동

성장한 캐릭터를 GitHub 프로필에 표시하여 자신의 코딩 활동을 시각적으로 표현할 수 있습니다.

- GitHub 프로필에 캐릭터 표시
- 현재 캐릭터 상태 반영
- 개성 있는 개발자 프로필 제공

---

### User Flow

```text
회원가입 및 로그인
        ↓
GitHub 및 코딩 테스트 플랫폼 계정 연동
        ↓
백준 / 프로그래머스 문제 해결
        ↓
문제 해결 기록 기반 코인 획득
        ↓
상점에서 아이템 구매
        ↓
캐릭터 육성 및 꾸미기
        ↓
GitHub 프로필에 캐릭터 표시
```

## Scope

### 구현 범위

- 회원가입 및 로그인
- 코딩 테스트 플랫폼 계정 연동
- 문제 해결 기록 확인
- 난이도 및 풀이 시간 기반 코인 지급
- 음식 및 의상 구매
- 캐릭터 상태 변화
  - 배고픔
  - 행복도
  - 체형
- GitHub 프로필 캐릭터 표시

### Out of Scope

초기 버전에서는 아래 기능은 구현하지 않습니다.

- 친구 추가
- 실시간 채팅
- 사용자 랭킹
- 캐릭터 결혼
- 공동 집 꾸미기
- 반려동물
- 푸시 알림
- 방 꾸미기
- 가구 꾸미기
- 계절 이벤트
- 복잡한 운동 및 건강 관리 시스템

핵심 기능인

- 코딩 기록 연동
- 코인 획득
- 음식 먹이기
- 의상 꾸미기
- GitHub 프로필 표시

기능 구현에 집중합니다.
